# COMP3000
A repository for my COMP3000 final year project:

A security and privacy prioritising geofencing application with a wide range of configuration options that do not comprimise on the key principles of cyber-security.

This relies on homomorhpic encryption, done using Microsoft's SEAL library to preserver data privacy during computation.

As this is cloud-hosted, the web UI can be viewed at https://cybertrack.azurewebsites.net/

To use the Android application, open the /SourceCode/Android folder in Android Studio, and run using an emulator with Android API 36.1 or older.
